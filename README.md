
# How to Run Project
1, 2, 3, 4 correspond to 
adaptiveA, backwardA, forwardA largestG, forward smallestG searches

In the terminal, run
```
python maze.py [number] .\testcases\testcase[number]
```
For example, 
To run adaptiveA on testcase1:
```
python maze.py 1 .\testcases\testcase1
```
